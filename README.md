# -simula-o-de-investimentos-em-fundos-imobili-rios
Projeto desenvolvido na DIO com foco na criação de uma planilha em Excel para simulação de investimentos em Fundos Imobiliários (FIIs). A ferramenta automatiza cálculos de valor investido, patrimônio acumulado e dividendos mensais, auxiliando na tomada de decisão e documentando todo o processo técnico no GitHub.
